## Installations

#### Requirements

This project was created using following library versions:

* Python3 (3.12.3)

* numpy (1.26.0)
* tabulate (0.9.0)
* matplotlib (3.9.2)
* shapely (2.0.6)
* pygame (2.0.1)
* scikit-learn (1.5.2)
* numba (0.60.0)

Install the requirements using:
```
pip install -r requirements.txt
```

#### Install
```
git clone
```
